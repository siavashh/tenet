Tenet, is a modern, responsive and customizable yet free WordPress theme.

== Description ==

Tenet has multiple settings to change the view of the website, including boxed and unBoxed, multi column and no sidebar setting, and also full color scheme settings, with some default built-in color schemes, social media buttons and many more to come...

== Installation ==
Manual installation:

1. Upload the `tenet` folder to the `/wp-content/themes/` directory

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Tenet Options to change theme specific options

== Custom Logo ==
This theme supports custom logo for the header part. 
You can change or remove it from Appearance -> Customize.
